# คู่มือการใช้งาน — AI Coding Adventure

## สำหรับนักเรียน

### เข้าสู่ระบบ

1. เปิดเบราว์เซอร์ → ไปที่ลิงก์ที่ครูให้มา
2. ใส่ **รหัสนักเรียน** และ **รหัสผ่าน** ที่ได้รับจากครู
3. กดปุ่ม **เข้าสู่ระบบ**
4. ระบบจะพาไปยังหน้าหลักนักเรียนโดยอัตโนมัติ

> **เคล็ดลับ**: เปิดบนโทรศัพท์หรือแท็บเล็ตได้เช่นกัน! กด "Add to Home Screen" เพื่อติดตั้งเป็นแอป

---

### หน้าหลัก (Home)

หน้าหลักแสดง:
- **ยินดีต้อนรับ** + ชื่อ, ระดับ, และ XP ปัจจุบัน
- **Mission วันนี้** — ภารกิจที่แนะนำสำหรับวันนี้
- **XP Progress Bar** — ความก้าวหน้าสู่ระดับถัดไป
- **ภารกิจล่าสุด** — รายการภารกิจที่ผ่านแล้ว
- **Badge ที่ได้รับ** — เหรียญรางวัลที่สะสม

---

### ทำ Mission

1. ไปที่เมนู **Mission** (หรือคลิก "เริ่มภารกิจ" จากหน้าหลัก)
2. เลือกภารกิจที่ต้องการทำ
3. อ่านเนื้อหาบทเรียนในแต่ละขั้นตอน
4. ตอบคำถาม/โจทย์ Quiz
5. กด **ตรวจคำตอบ** เพื่อเช็คคำตอบ
6. เมื่อผ่านทุกข้อ กด **ส่งคำตอบ**

**ระดับความยาก:**
- 🟢 Beginner — Mission 1, 2
- 🟡 Intermediate — Mission 3, 4
- 🔴 Advanced — Mission 5

**เงื่อนไขผ่าน**: ได้คะแนน ≥ 60%

**XP ที่ได้รับ**: คำนวณตามสัดส่วนคะแนน (เช่น 80% ของ 100 XP = 80 XP)

---

### Badge และ XP

**ระดับ (Level):**

| Level | ชื่อ | XP ที่ต้องการ |
|-------|------|--------------|
| 1 | มือใหม่ | 0 |
| 2 | ผู้เรียนรู้ | 300 |
| 3 | นักสำรวจ | 600 |
| 4 | นักพัฒนา | 1,000 |
| 5 | AI Master | 1,500 |

**Badge ที่สะสมได้:**

| Badge | วิธีได้รับ |
|-------|------------|
| mission1_complete | ผ่าน Mission 1 |
| mission2_complete | ผ่าน Mission 2 |
| mission3_complete | ผ่าน Mission 3 |
| mission4_complete | ผ่าน Mission 4 |
| mission5_complete | ผ่าน Mission 5 |
| perfect_score | ได้คะแนน 100% ในภารกิจใด ๆ |
| speed_learner | ทำ Mission สำเร็จภายใน 15 นาที |
| portfolio_first | อัปโหลดผลงานชิ้นแรก |
| all_missions | ผ่านครบทั้ง 5 Mission |

---

### Portfolio

1. ไปที่เมนู **Portfolio**
2. กดปุ่ม **+ อัปโหลดผลงาน**
3. ใส่ชื่อผลงาน, เลือก Mission ที่เกี่ยวข้อง, คำอธิบาย
4. เลือกไฟล์ (รองรับ PNG, JPG, GIF, PDF ขนาดสูงสุด 10 MB)
5. กด **บันทึกผลงาน**

ผลงานจะแสดงในกริดและสามารถกดดูรายละเอียดได้

---

### Leaderboard

ดูอันดับห้องเรียนได้จากเมนู **อันดับ**:
- อันดับ 1-3 แสดงบน Podium
- ตารางอันดับเต็มด้านล่าง
- แถวสีทองคือตำแหน่งของคุณเอง

---

### โปรไฟล์

ไปที่เมนู **โปรไฟล์** เพื่อ:
- แก้ไขชื่อ, ชื่อเล่น, อีเมล, คำแนะนำตัว
- เปลี่ยนสี Avatar (8 สีให้เลือก)
- เปลี่ยนรหัสผ่าน
- ดูความก้าวหน้า Mission ทั้งหมด
- ดู Badge Collection

---

## สำหรับครู

### เข้าสู่ระบบครู

1. เปิดเบราว์เซอร์ → ไปที่ลิงก์ที่ได้รับ
2. ใส่ **รหัสครู** และ **รหัสผ่าน**
3. ระบบจะพาไปยัง **Teacher Dashboard** โดยอัตโนมัติ

---

### Teacher Dashboard

Dashboard แสดง:

**สถิติภาพรวม (4 การ์ดด้านบน):**
- จำนวนนักเรียนทั้งหมด
- จำนวนภารกิจที่ผ่านรวม
- XP เฉลี่ยของนักเรียน
- จำนวน Badge ที่มอบแล้ว

**ตารางอันดับนักเรียน:**
- เรียงตาม XP สูง → ต่ำ
- แสดงระดับ, XP, จำนวน Mission, และ Badge ของแต่ละคน

**กราฟ XP ต่อภารกิจ:**
- แสดงเปอร์เซ็นต์นักเรียนที่ผ่านแต่ละ Mission

**ความคืบหน้าภารกิจ:**
- Progress bar แต่ละ Mission

## คำถามที่พบบ่อย

**Q: ลืมรหัสผ่านทำอย่างไร?**  
A: ติดต่อครูเพื่อให้ Reset รหัสผ่านใน Google Sheets

**Q: ทำ Mission ซ้ำได้ไหม?**  
A: ทำซ้ำได้ แต่ XP จะไม่ได้รับอีก (ได้เฉพาะครั้งแรกที่ผ่าน)

**Q: ไฟล์ Portfolio หายไปไหน?**  
A: ไฟล์เก็บใน Google Drive ของโรงเรียน ข้อมูลจะแสดงตราบเท่าที่ยังล็อกอินอยู่

**Q: ใช้บนมือถือได้ไหม?**  
A: ได้ — รองรับทุก device ขนาดหน้าจอ

**Q: ต้องติดตั้งอะไรไหม?**  
A: ไม่ต้อง — ใช้งานผ่านเบราว์เซอร์ได้เลย (รองรับ Chrome, Edge, Safari, Firefox)
